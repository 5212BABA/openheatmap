OpenGraphMap
~~~~~~~~~~~~

OpenGraphMap is a system for creating, displaying and interacting with data visualizations based on geographic data. Written by Pete Warden <pete@petewarden.com>, all code is licensed under the GPL V3. For more details on the license see the included gpl.txt file or go to http://www.gnu.org/licenses/

It's not part of the OpenStreetMap project, but it does use the OSM XML format for storing geographic data.

For detailed help and information, go to http://opengraphmap.org/
